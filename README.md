# Teach Me

Curated, in-depth frontend lessons for developers and designers.

## Courses

- **[Kiro Mastery](/kiro/)** - 16 lessons covering steering, context management, hooks, skills, MCP servers, powers, sub-agents, custom agents, ACP, property-based testing, specs, model selection, multi-root workspaces, and CLI workflows.
- **[CSS Mastery](/css/)** - 24 lessons covering custom properties, @property, value functions, flexbox, grid, container queries, cascade layers, nesting, modern selectors, view transitions, anchor positioning, scroll-driven animations, and more.
- **[SVG - From Coordinates to Animation](/svg/)** - 13 lessons covering the coordinate system, shapes, paths, styling, gradients, filters, text, SMIL animation, CSS animation, and JavaScript interaction.

## About

Free, reference-grade tutorials built for developers who value precise, visual explanations. Content created with AI assistance and manually verified.
